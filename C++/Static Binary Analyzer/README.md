It is a tool that inspects compiled C++ programs without executing them. Its main purpose it to uncover potential issues such as security vulnerabilites, bugs and non compliance with coding standards. Its key characteristics include :
a) Non execution based
b) Binary level focus
c) Root cause identification
d) Used in malware detection , finding embedded secrets and strengthening software supply chain security
e) Techniques : disassembly , decompilation, control flow graph construction, data flow analytics
f) C++ specific benefits
